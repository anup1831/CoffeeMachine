Cubing even numbers and doubling others

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a function in Kotlin that receives a positive integer as input and returns the cube of that number if it's even, in all other cases it should return twice the number. Ignore negative numbers and zero.
Sample Input:
2

Sample Output:
8


Sample Input:
3

Sample Output:
6



Memory limit: 256 MB
Time limit: 8 seconds
